## Welcome!

This is your first blog post. It lives in `content/blog/hello-world.md` — open it in your editor and start writing.

### How to add a new post

Create a new `.md` file in the `content/blog/` folder with frontmatter at the top:

```markdown
---
title: "Your Post Title"
date: "2026-01-20"
summary: "A short description of your post."
tags: ["tag1", "tag2"]
---

Your content here...
```

That's it. The blog will pick it up automatically.

### What's supported

- **Bold**, *italic*, ~~strikethrough~~
- [Links](https://example.com)
- Images, blockquotes, tables
- Code blocks with syntax hints
- Lists (like this one!)
